Welcome to **The Shadow Index**! 👋

## 🚀 Interactive Setup Checklist

Let's make sure everything is ready for this mission. Check these off as you go!

- [ ] Open the terminal
- [ ] Install dependencies with `npm install`
- [ ] Run the development server with `npm run dev`
- [ ] Open [http://localhost:3000](http://localhost:3000) in your top-secret browser
- [ ] Modify `app/(dashboard)/page.tsx` and watch it instantly update!

## 🛠️ How to run the project

Getting started is easy. Just run this command:

```bash
npm run dev
```

Then visit [http://localhost:3000](http://localhost:3000).
